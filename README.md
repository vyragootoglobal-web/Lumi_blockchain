# Lumi_blockchain


Lumi adalah blockchain publik yang dikelola komunitas. Repo ini berisi node scripts v1-v20, utilities, dan dokumentasi.

## Cara Memulai
1. Install Python 3.8+
2. Clone repository
3. Masuk ke folder `nodes/`
4. Jalankan node:
   ```bash
   python node_v1.py
